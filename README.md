# Mangata Gallo

## Description

Mangata Gallo is a front-end web development portfolio project representing a high-quality jewelry store. The website features a modern, responsive design that showcases products like rings, bracelets, and earrings. It demonstrates the use of semantic HTML, modern CSS styling, and responsive layout techniques.

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop screens using CSS media queries.
- **Glassmorphism UI**: Features a distinct visual style with semi-transparent backgrounds and blur effects.
- **Product Showcase**: A grid layout displaying various jewelry items with images and prices.
- **Contact Integration**: A functional contact form integrated with Formspree.
- **Navigation**: Adaptive navigation menu that switches from a vertical stack on mobile to a horizontal bar on desktop.

## Technologies Used

- **HTML5**: For structure and semantics.
- **CSS3**: For styling, including Flexbox, CSS Grid, and CSS Variables.
- **JavaScript**: For interactive elements.

## Setup & Usage

To view the project locally:

1. Clone or download this repository.
2. Navigate to the project folder.
3. Open `index.html` in any standard web browser.

## Project Structure

- `index.html`: The main landing page.
- `Products.html`: The product catalog page.
- `About.html`: Information about the brand.
- `Contact.html`: The contact form page.
- `Cart.html`: The shopping cart interface.
- `style.css`: The main stylesheet containing all design rules.
- `Script.js`: JavaScript file for site functionality.
- `Media/`: Directory containing images and assets.
